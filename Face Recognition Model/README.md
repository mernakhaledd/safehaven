# SafeHaven: Lightweight Face Recognition Demo

This is a standalone, lightweight face recognition system optimized for environments without a GPU, such as a Raspberry Pi. It takes input from a webcam, detects faces in real-time, matching them with a known database or marking them as "Unknown".

## 1. Model Architecture Overview

- **Detection Method:** [OpenCV YuNet](https://github.com/opencv/opencv_zoo/tree/main/models/face_detection_yunet).
  - *Why?* A highly efficient CNN-based face detector built directly into OpenCV 4.x. It runs blazingly fast purely on the CPU without requiring heavy frameworks.
- **Recognition Method:** [OpenCV SFace](https://github.com/opencv/opencv_zoo/tree/main/models/face_recognition_sface).
  - *Why?* It generates a 128-dimensional embedding for faces similar to FaceNet, but is built entirely on native OpenCV DNN. **Crucially, it requires ZERO extra dependencies like TensorFlow, PyTorch, or C++ wrappers (dlib)**. It executes directly via `opencv-python`.
- **Distance Comparison:** Cosine distance threshold matching (set at `0.36` strictly, but tweakable).

## 3. How to Run (Step-by-Step)

### Step 1: System Setup
Since we upgraded the logic to native OpenCV Deep Learning models, there are **NO C++ compilation requirements**. It's purely Python:
\`\`\`bash
pip install -r requirements.txt
\`\`\`

### Step 2: Enrollment (Adding Faces)
1. In the `dataset/known_faces` directory, create a folder for each person you want to know (e.g., `dataset/known_faces/Alice`).
2. Put one or more clear pictures of Alice's face inside her folder.
3. Run the enrollment script:
\`\`\`bash
python enroll.py
\`\`\`
*(It will extract structural embeddings of the faces and save them to `embeddings.pkl`)*

### Step 3: Run the Real-Time Demo
1. Make sure your webcam is attached/enabled.
2. Run the detection loop:
\`\`\`bash
python recognize.py
\`\`\`
*(A window will pop up drawing bounding boxes over faces. Detected matches display their names; otherwise, they appear as "Unknown")*

### Step 4: Run the Evaluation (Accuracy Testing)
1. Add test images (that were *not* used in training) to the `dataset/test_faces/` folder. Use folder names corresponding to names you expect the model to guess (or "Unknown" folder for random strangers).
2. Run the accuracy evaluation:
\`\`\`bash
python evaluate.py
\`\`\`
*(It will print a breakdown of all matches vs mismatches and an accuracy score.)*
